# 🌐 Bilingual UI Update — ไทย & English

## ✨ การปรับปรุงด้าน UI/UX

### 📝 ภาษาอังกฤษ
- เพิ่ม **English labels** ใต้ทุกเมนูภาษาไทย
- ตัวอักษร **เล็กกว่า** (10-11px) เพื่อไม่ให้เด่นเกินไป
- ใช้สี **#AAA (สีเทาอ่อน)** + opacity 0.8 เพื่อ subtle appearance

